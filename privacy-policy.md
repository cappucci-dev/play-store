Política de Privacidade

Última atualização: 13/04/2026

Este aplicativo ("App") foi desenvolvido para fornecer entretenimento e/ou utilidade aos usuários, respeitando sua privacidade.

1. Coleta de informações
O App pode coletar algumas informações fornecidas pelo usuário de forma opcional, como:
- Nickname (apelido)
- País
- Estado

Essas informações são utilizadas exclusivamente para funcionalidades internas do aplicativo, como rankings e classificação de jogadores.

Além disso, serviços de terceiros podem coletar informações automaticamente, como:
- Endereço IP
- Tipo de dispositivo
- Sistema operacional
- Dados de uso do aplicativo

2. Uso de serviços de terceiros
Este App utiliza serviços de terceiros para funcionamento e melhoria da experiência, incluindo:

- Firebase (Google): utilizado para armazenamento e gerenciamento de dados, como rankings de usuários.
- Google AdMob: utilizado para exibição de anúncios.

Esses serviços podem coletar e processar dados conforme suas próprias políticas de privacidade.

Para mais informações:
https://policies.google.com/privacy

3. Uso das informações
As informações coletadas são utilizadas para:
- Criar e exibir rankings de jogadores
- Melhorar a experiência do usuário
- Exibir anúncios relevantes
- Analisar o uso do aplicativo

4. Compartilhamento de informações
O App não vende, troca ou compartilha informações pessoais dos usuários com terceiros, exceto quando necessário para funcionamento dos serviços utilizados (como Firebase e AdMob).

5. Armazenamento e segurança
Os dados podem ser armazenados em servidores seguros fornecidos por serviços de terceiros. Embora sejam adotadas medidas de segurança, nenhum método de transmissão ou armazenamento é 100% seguro.

6. Direitos do usuário
O usuário pode optar por não fornecer informações opcionais. Caso deseje a remoção dos dados, poderá entrar em contato através do e-mail informado abaixo.

7. Alterações nesta política
Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página regularmente.
